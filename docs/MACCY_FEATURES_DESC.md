# Maccy 功能详细描述 (基于截图分析)

## 1. 通用 (General)
- **系统集成**：支持登录时自启动，支持自动或手动检查更新。
- **全局快捷键管理**：
  - `Open`: 呼出主界面的核心热键。
  - `Pin`: 快速固定选中条目的热键。
  - `Delete`: 快速删除选中条目的热键。
- **搜索算法**：支持“精确匹配”与“模糊搜索”切换。
- **自动化行为**：
  - `Auto-paste`: 选中即粘贴的无缝体验。
  - `Plain text`: 强制以纯文本格式粘贴。
  - **组合键修饰**：支持使用 Option/Command/Shift 等修饰键改变点击时的默认行为（仅复制、复制+粘贴、复制+匹配格式粘贴）。

## 2. 存储 (Storage)
- **内容过滤**：可选择性保存文本、图像或文件路径。
- **存储策略**：
  - 数量限制（默认 200 条）。
  - 排序逻辑（按最后复制时间）。
- **资源监控**：实时显示当前历史记录占用的磁盘/内存空间。

## 3. 外观 (Appearance)
- **窗口定位**：支持跟随光标（Cursor）、居中（Center）等弹出模式。
- **固定策略**：固定项目可置顶或置底显示。
- **多媒体支持**：可配置图片预览的高度及弹出延迟。
- **界面元素开关**：
  - 显示特殊字符。
  - 自定义菜单栏图标样式（剪贴板图标等）。
  - 在列表项中显示来源应用图标。
  - 搜索框内显示应用名称。
  - 底部功能菜单显隐。

## 4. 固定 (Pins)
- **静态条目管理**：用户可手动创建“永久固定”的短语或代码片段。
- **自定义映射**：为每个固定项分配唯一的快捷键、显示标题和原始内容。

## 5. 忽略 (Ignore)
- **应用黑名单**：防止从敏感应用（如密码管理器）中读取剪贴板。
- **类型过滤**：忽略特定的系统剪贴板类型。
- **内容过滤**：使用正则表达式（Regex）自动忽略匹配到的敏感文本（如信用卡号）。

## 6. 高级 (Advanced)
- **运行控制**：一键暂停记录。
- **隐私保护**：
  - 应用退出时自动清空历史。
  - 清空历史时同步抹除系统剪贴板，防止残留。
- **开发者集成**：提供 CLI 命令（defaults write...）用于脚本化控制。
