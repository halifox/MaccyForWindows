## 4. 测试核心功能

### 测试 1: 自动粘贴
1. 打开记事本
2. 复制几段不同的文本
3. 按 `Alt+V` 打开 Maccy
4. 选择一个历史记录按 `Enter`
5. ✅ 验证：文本自动粘贴到记事本

### 测试 2: 循环选择模式
1. 确保有至少 5 条历史记录
2. **按住 `Alt` 键不放**
3. **快速按 3 次 `V`**
4. ✅ 验证：窗口打开并自动选中第 3 项
5. 继续按 `V` 2 次（仍然按住 `Alt`）
6. ✅ 验证：选中第 5 项
7. **松开 `Alt` 键**
8. ✅ 验证：第 5 项自动粘贴到记事本

### 测试 3: 窗口定位
1. 移动鼠标到屏幕右下角
2. 按 `Alt+V`
3. ✅ 验证：窗口在光标附近打开（不超出屏幕边界）

### 测试 4: 修饰键释放检测
1. 按住 `Alt`，按 `V` 打开窗口
2. 继续按住 `Alt`，按 `↓` 选择下一项
3. 松开 `Alt`
4. ✅ 验证：选中项自动粘贴

## 5. 配置选项

在应用设置中可以配置：

### 弹窗位置
- `cursor` - 光标跟随（默认）
- `center` - 屏幕中心
- `statusItem` - 状态栏图标
- `lastPosition` - 记住位置

### 自动粘贴
- 开启（默认）：选择后自动粘贴
- 关闭：仅复制到剪贴板

### 多屏幕
- `0` - 活动屏幕（光标所在）
- `1+` - 具体屏幕编号

## 6. 常见问题

### Q: 粘贴没有反应？
A: 检查目标应用是否支持 `Ctrl+V`，某些应用可能需要更长延迟。

### Q: 循环模式不工作？
A: 确保快速连续按键，两次按键间隔不超过 200ms。

### Q: 窗口位置不对？
A: 检查 `popupPositionProvider` 配置，确认屏幕索引正确。

### Q: 修饰键检测延迟？
A: 默认 50ms 轮询，可在 `modifier_key_service.dart` 中调整。

## 7. 性能优化建议

1. **减少历史记录数量**：设置 → 存储 → 历史记录上限（默认 200）
2. **关闭不需要的类型**：设置 → 存储 → 取消勾选图片/文件
3. **调整检查间隔**：减少剪贴板轮询频率

## 8. 调试模式

查看详细日志：
```bash
flutter run -d windows --verbose
```

关键日志标记：
- `[HotKeyManager]` - 热键和状态机
- `[WindowManager]` - 窗口定位
- `[ClipboardManager]` - 剪贴板和粘贴
- `[ModifierKeyService]` - 修饰键检测
